Alpha Distribution

Alpha Distribution is a comprehensive e-commerce web application built with Laravel, designed to streamline product distribution and sales management. The system includes two separate interfaces: Admin and Salesman, each tailored to their specific roles and responsibilities.

Features
Admin Panel

Product Management: Add, edit, and delete products

Category Management: Organize products into categories

Order Management: View and manage all orders, print vouchers

Way Plan Management: Assign daily routes and tasks for salesmen

User Management: Add, remove, and organize users

Reporting: Generate daily sales reports and export/print them

Salesman Panel

Secure Login: Access via username and password

Outlet Management: View assigned outlets based on todayâ€™s way plan

Product Ordering: Select products to create temporary orders

Order Upload: Submit completed sales data for admin review

Order List: Track and manage all orders before submission

Technology Stack

Backend: Laravel

Frontend: Blade Templates, HTML, CSS, JavaScript

Database: MySQL

Authentication & Authorization: Laravel built-in auth system

How It Works

Admin Workflow:

Logs in to manage products, users, and way plans

Monitors orders and daily sales reports

Prints vouchers for completed orders

Salesman Workflow:

Logs in to view assigned outlets for the day

Places orders for products from assigned outlets

Uploads completed sales data to the system
